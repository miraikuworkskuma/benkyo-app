# 📚 勉強アプリ

Googleドライブの教材(Googleドキュメント/スライド/PDF/画像)を読み込み、AIが**理解度チェック用の4択クイズ**と**暗記用フラッシュカード**を自動生成する学習アプリです。

- サーバー不要の静的Webアプリ。GitHub Pages などで**無料**でホスティングできます
- AIは **Gemini API**(各利用者が自分の無料キーを使用)。運営者の費用はゼロ
- APIキーや学習履歴は各利用者のブラウザ内(localStorage)にのみ保存されます

## 仕組み

```
利用者のブラウザ
 ├─ Googleログイン(OAuth) → 自分のDriveの教材(ドキュメント/スライド/PDF/画像)を読み込み
 ├─ 自分のGemini APIキーで問題を生成
 └─ クイズ/フラッシュカードで学習。履歴はブラウザに保存
```

## 管理者(あなた)のセットアップ手順

### 1. Google Cloud で OAuth クライアントIDを作る(無料・約10分)

1. [Google Cloud Console](https://console.cloud.google.com/) で新しいプロジェクトを作成(名前は自由。例: benkyo-app)
2. 「APIとサービス」→「ライブラリ」で **Google Drive API** を検索して「有効にする」
3. 「APIとサービス」→「OAuth同意画面」
   - User Type: **外部** を選択
   - アプリ名・メールアドレスを入力して保存
   - スコープの追加は不要(アプリ側で指定します)
   - **テストユーザー** に自分と友人のGmailアドレスを追加(最大100人)
4. 「APIとサービス」→「認証情報」→「認証情報を作成」→「OAuthクライアントID」
   - アプリケーションの種類: **ウェブアプリケーション**
   - 「承認済みのJavaScript生成元」に以下を追加:
     - ローカル確認用: `http://localhost:8765`
     - 公開用(後述のGitHub PagesのURL): `https://<ユーザー名>.github.io`
5. 発行された **クライアントID**(`xxxx.apps.googleusercontent.com`)を [config.js](config.js) の `GOOGLE_CLIENT_ID` に貼り付ける

### 2. GitHub Pages で公開する(無料)

1. GitHubで新しいリポジトリを作成(Public)
2. このフォルダのファイル一式をpush
3. リポジトリの Settings → Pages → Branch を `main` にして保存
4. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます
5. このURLをOAuthクライアントの「承認済みのJavaScript生成元」に追加し忘れないこと(パスは含めず `https://<ユーザー名>.github.io` まで)

### 3. 友人に伝えること

以下の2ステップとアプリのURLを共有すれば使えます。

1. 管理者にGmailアドレスを伝えてテストユーザーに登録してもらう
2. [Google AI Studio](https://aistudio.google.com/apikey) で無料のGemini APIキーを発行し、アプリ初回起動時の設定画面に貼り付ける(クレジットカード不要)

## 利用者の使い方

1. アプリのURLを開く
2. 初回のみ: Gemini APIキーとクライアントIDを設定画面に入力(config.jsにクライアントIDが設定済みならAPIキーだけ)
3. 「Googleドライブに接続」→ 自分のGoogleアカウントでログイン
4. 教材を選ぶ → モード(理解度クイズ / 暗記カード)と問題数を選んで「生成する」
5. 学習! 結果と教材セットはホームの「これまでの学習」から何度でも復習できます

## ローカルでの動作確認

```sh
python3 -m http.server 8765
# → http://localhost:8765 を開く
```

## 注意事項

- Gemini無料枠には回数制限があります(2.5 Flashで1分15回・1日あたり上限)。上限に達したら少し待ってください
- 無料枠では送信データがGoogleのモデル改善に使われる可能性があります。機密資料には使わないでください
- OAuth同意画面が「テスト」状態のままの場合、テストユーザーに登録した人だけがログインできます
- PDF・画像教材は1ファイル15MBまで読み込めます(スキャン教材もAIが直接読み取ります)
